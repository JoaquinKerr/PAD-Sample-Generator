# PAD Sample Generator
A simple python script to generate PAD synthesis samples along with an sfz file so you can load them into various samplers.
It doesn't have an interface. You have the edit it to get different results. It will generate a bland random sound by default, but you should tamper with the harmonics table and bandwitch parameters to home in on something with more charachter.

Read the original paper to understand what it does: https://zynaddsubfx.sourceforge.io/doc/PADsynth/PADsynth.htm
Good luck.

Also, check out my PAD Fabrics repository for some example patches for Shortcircuit XT.
